# MPNet
Implementation of Motion Planning Networks ([https://arxiv.org/abs/1806.05767](https://arxiv.org/abs/1806.05767)).

## Usage
### Training Models
1. train CAE
```
python CAE.py
```
2. train Planning Network
```
python PNet.py
```

### Generate Trajectory
```
python MPNet.py
```
